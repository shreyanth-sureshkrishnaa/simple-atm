# Student Bank ATM

A simple Python script which demonstrates how a basic ATM works. It handles the standard operations like checking your balance, depositing cash, and withdrawing money without going into the negative.

I wrote this mainly for my OOPs class, to practice working with classes and custom exception handling in Python.

## What's inside?

- **PIN Security**: It asks for a 4-digit PIN.
- **Transaction History**: Keeps a log of what you've done during the session.

## How to run it

```bash
python main.py
```
