# Student Bank ATM

Just a simple Python script I put together to simulate how a basic ATM works. It handles the usual stuff like checking your balance, depositing some cash, and withdrawing money without going into the negative.

I wrote this mainly for my OOPs class, to practice working with classes and custom exception handling in Python.

## What's inside?

- **PIN Security**: It asks for a 4-digit PIN before you can do anything.
- **Balance Checks**: Tells you exactly how much you have left.
- **Deposits & Withdrawals**: Standard.
- **Transaction History**: Keeps a log of what you've done during the session.

## How to run it

```bash
python main.py
```
